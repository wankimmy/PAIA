# Contributing to Personal AI Assistant

Thank you for your interest in contributing!

## Development Setup

1. Fork the repository
2. Clone your fork
3. Follow the setup instructions in SETUP.md
4. Create a feature branch
5. Make your changes
6. Test thoroughly
7. Submit a pull request

## Code Style

- **Backend**: Follow Laravel coding standards (use Laravel Pint)
- **Frontend**: Follow Vue 3 style guide and use ESLint

## Testing

- Write tests for new features
- Ensure all tests pass before submitting PR

## Commit Messages

Use clear, descriptive commit messages following conventional commits format.

## Security

- Never commit sensitive data (API keys, passwords, etc.)
- Report security vulnerabilities privately
